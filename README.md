# material-music

一个material风格的云音乐平台，使用flex做了响应式布局

在线演示地址：[material-music](http://cloud.huajiayi.top/)

## 技术栈
前端：React + react-router-dom + Redux + Axios

后端：Spring Boot + Spring MVC + MyBatis + Shiro

UI：Material-UI

## 功能
### 用户模块
- 用户注册
- 用户登录
- 用户信息修改

### 音乐模块
- 最新音乐
- 播放音乐
- 添加到歌单

### 歌单模块
- 推荐歌单
- 我的歌单
- 新建歌单
- 歌单详情

### 评论模块
- 评论歌单
- 评论点赞

## 特性
- 换肤功能
- 使用骨架屏代替传统loading
- 图片懒加载
- 使用Redis缓存登录用户信息
- 基于GitHub Actions的工作流，实现了自动部署
